Questions fréquentes

L'agent peut-il évaluer la faisabilité d'un projet ?

Non. L'évaluation de faisabilité est un acte professionnel qui engage votre responsabilité. L'agent collecte les informations nécessaires (surface, terrain, PLU, budget) et prépare la fiche pour votre analyse. Il peut répondre à des questions générales sur les réglementations courantes (bande de recul, hauteur maximale standard), mais toute étude spécifique reste de votre ressort.

Comment l'agent gère-t-il les délais administratifs ?

L'agent peut être configuré avec les délais d'instruction standard (2 mois pour une déclaration préalable, 3 mois pour un permis de construire, délais spécifiques en secteur protégé) et informer les clients de façon générale. Pour le suivi de leur dossier spécifique, il notifie selon l'avancement que vous lui communiquez.

L'agent peut-il m'aider sur les appels d'offres et marchés publics ?

L'agent peut surveiller les plateformes de publication (BOAMP, e-marchés publics) selon des critères que vous définissez (localisation, type de marché, montant), alerter votre équipe et collecter les documents de candidature. La rédaction des mémoires techniques reste humaine.

Compatible avec les logiciels de gestion de cabinet (ArchiOffice, Archicad, Domus) ?

L'agent s'intègre à votre agenda et à votre CRM via API ou export. Pour ArchiOffice, une synchronisation des projets est possible pour le suivi client. La partie BIM et conception reste hors du périmètre de l'agent.

En combien de temps est-il opérationnel ?

7 jours ouvrés pour un agent standard (qualification prospects, FAQ réglementaire, relances). Vous définissez les types de projets que vous acceptez, vos zones géographiques et vos honoraires indicatifs pour les projets standards.
