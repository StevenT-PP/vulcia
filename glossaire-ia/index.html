Référence IA 2026

Glossaire IA pour
dirigeants de PME

40+ termes expliqués sans jargon. De "agent IA" à "webhook" — tout ce qu'il faut savoir pour prendre des décisions éclairées sur vos projets IA.

ACDEFGHIJLMNOPRSTW
A

Agent IA

Programme autonome capable de comprendre des instructions en langage naturel, de prendre des décisions et d'agir sur des systèmes externes (agenda, email, CRM) sans intervention humaine à chaque étape. Différent d'un chatbot simple qui suit des règles fixes.

API (Interface de Programmation)

Protocole permettant à deux logiciels de communiquer. Un agent IA utilise des APIs pour accéder à votre agenda, envoyer des emails ou lire votre base clients. L'intégration API est ce qui transforme un modèle IA en outil métier utile.

Automatisation IA

Exécution automatique de tâches répétitives par un système IA sans intervention humaine. Exemples : réponse aux emails, prise de RDV, envoi de devis, relance clients. Différente de l'automatisation classique (règles) : elle comprend le contexte.

AI Overview (Google)

Encadré généré par l'IA de Google qui apparaît en haut des résultats de recherche. Pour y figurer, votre contenu doit être structuré, factuel, et répondre directement à la question posée. Correspond à l'enjeu GEO (Generative Engine Optimization).

C

ChatGPT

Modèle de langage d'OpenAI accessible au grand public. Puissant pour des usages génériques (rédaction, résumé, code). Non configuré pour votre activité sans personnalisation. La version "Team" ou les GPT Custom permettent un début de configuration métier.

Claude (Anthropic)

Modèle de langage d'Anthropic, concurrent de GPT-4o. Reconnu pour la qualité de sa rédaction et son respect des instructions complexes. Hébergé aux États-Unis, ce qui impose des précautions RGPD pour les données sensibles.

CLS (Cumulative Layout Shift)

Métrique Core Web Vitals mesurant la stabilité visuelle d'une page. Un score CLS élevé signifie que les éléments de la page bougent pendant le chargement, dégradant l'expérience utilisateur. Score idéal : < 0,1. Facteur de classement Google depuis 2021.

Contexte (fenêtre de contexte)

Quantité de texte que le modèle IA peut "voir" et comprendre en une seule interaction. GPT-4o : 128 000 tokens (~100 000 mots). Plus le contexte est grand, plus l'IA peut analyser des documents longs ou maintenir une conversation longue sans oublier le début.

Core Web Vitals (CWV)

Les trois métriques de performance web retenues par Google comme facteurs de classement SEO : LCP (chargement), INP (interactivité) et CLS (stabilité visuelle). Un site passant les seuils Google est récompensé dans les résultats de recherche.

CRM (Customer Relationship Management)

Logiciel de gestion de la relation client (HubSpot, Salesforce, Pipedrive…). Un agent IA peut alimenter votre CRM automatiquement : créer une fiche prospect, noter les échanges, mettre à jour le statut d'un deal après chaque interaction.

D

Données d'entraînement

Corpus de texte sur lequel un modèle IA a été entraîné. GPT-4o a été entraîné sur des milliards de pages web jusqu'à une certaine date. Ces données déterminent les connaissances "natives" du modèle, indépendamment de la configuration métier ajoutée ensuite.

E

E-E-A-T (Experience, Expertise, Authoritativeness, Trustworthiness)

Cadre d'évaluation de la qualité de contenu utilisé par les évaluateurs humains de Google. En 2026, l'Experience (expérience terrain de l'auteur) est le critère le plus discriminant. Un contenu signé par un expert identifiable avec preuves d'expérience est mieux classé.

Embedding

Représentation numérique d'un texte sous forme de vecteur de nombres. Permet à l'IA de mesurer la similarité sémantique entre deux textes. Utilisé dans les systèmes RAG pour retrouver les passages pertinents dans une base de données.

Escalade (agent IA)

Mécanisme par lequel un agent IA transfère une demande à un humain quand elle dépasse son périmètre ou sa confiance. Une bonne configuration d'escalade est critique : l'agent doit savoir quand il ne sait pas.

F

Fine-tuning

Entraînement supplémentaire d'un modèle IA sur des données spécifiques pour améliorer ses performances sur un domaine précis. Coûteux en temps et données. Alternative plus légère et souvent suffisante : le prompt engineering ou le RAG.

Foundation Model

Modèle IA de grande taille entraîné sur des volumes massifs de données générales (GPT-4o, Claude, Mistral Large). Sert de base à des applications spécialisées via configuration, fine-tuning ou RAG. Distingué des modèles spécialisés plus petits.

G

GEO (Generative Engine Optimization)

Optimisation de votre contenu pour apparaître dans les réponses des moteurs de recherche génératifs (Google AI Overviews, ChatGPT Search, Perplexity, Bing Copilot). Différent du SEO classique : l'objectif est d'être cité, pas seulement d'être indexé.

GPT-4o

Modèle d'OpenAI ("o" pour omnimodal) capable de traiter du texte, des images et de l'audio. Standard de fait pour les agents IA polyvalents en 2026. Hébergé aux États-Unis. Coût par token plus élevé que Mistral pour les volumes importants.

Guardrails (garde-fous)

Contraintes ajoutées à un agent IA pour limiter ses comportements indésirables : ne pas inventer de prix, ne pas promettre des délais non confirmés, ne pas répondre hors du périmètre métier. La qualité des guardrails détermine la fiabilité de l'agent en production.

H

Hallucination

Phénomène où un modèle IA génère une information fausse mais présentée avec confiance. Les hallucinations diminuent avec les meilleurs modèles mais ne disparaissent pas. Solution : RAG (ancrage sur des données vérifiées) et validation humaine pour les informations critiques.

Hébergement souverain

Stockage des données sur des serveurs situés en France ou en Europe, sous juridiction française. Requis pour les données de santé (RGPD, HDS). Mistral AI (Paris) est l'option principale pour un hébergement IA souverain en France.

I

INP (Interaction to Next Paint)

Métrique Core Web Vitals mesurant la réactivité d'une page aux interactions utilisateur (clic, toucher, clavier). Remplace le FID depuis 2024. Seuil "bon" : < 200ms. Un score élevé signale un site avec trop de JavaScript bloquant.

Instruction System (System Prompt)

Instructions données en amont à un agent IA pour définir son comportement, son identité, son périmètre et ses limites. C'est la "carte d'identité" de l'agent. La qualité du system prompt est déterminante pour la pertinence des réponses.

J

JSON-LD

Format de données structurées recommandé par Google pour les balises Schema.org. Intégré dans le HTML via une balise `<script type="application/ld+json">`. Permet aux moteurs de recherche de comprendre le type de contenu (FAQ, Article, Produit, Service) et d'afficher des résultats enrichis.

L

LCP (Largest Contentful Paint)

Métrique Core Web Vitals mesurant le temps de chargement du plus grand élément visible à l'écran. Seuil "bon" : < 2,5 secondes. Principal signal de performance perçue par l'utilisateur. Un LCP > 4s entraîne un taux de rebond significativement plus élevé.

LLM (Large Language Model)

Modèle de traitement du langage naturel entraîné sur des milliards de textes. GPT-4o, Claude, Mistral sont des LLMs. Ils comprennent et génèrent du texte humain de manière convaincante. La base technique de tous les agents IA actuels.

llms.txt

Fichier texte placé à la racine d'un site (comme robots.txt) destiné aux crawlers des moteurs IA. Il décrit le site, ses services et ses données clés dans un format que les LLMs peuvent lire facilement. Recommandé pour le GEO.

M

Mistral Large

Modèle de langage de Mistral AI, entreprise française (Paris). Principal concurrent européen de GPT-4o et Claude. Avantage clé : peut être hébergé en France, conforme RGPD natif. Idéal pour les données sensibles (santé, juridique, RH).

Multimodal

Capacité d'un modèle IA à traiter plusieurs types de données : texte, images, audio, vidéo. GPT-4o est multimodal (analyse d'images, transcription audio). Mistral Large a des capacités multimodales limitées en comparaison.

N

No-show

Client qui ne se présente pas à un rendez-vous sans prévenir. Coût moyen en cabinet médical : 15 à 40 € par no-show. Les agents IA réduisent le taux de no-show de 50 à 70% grâce aux rappels automatiques SMS/email à J-1 et J-2.

NLP (Natural Language Processing)

Traitement automatique du langage naturel. Discipline de l'IA qui permet aux machines de comprendre, analyser et générer du texte humain. Les LLMs sont la forme la plus avancée de NLP à ce jour.

O

Orchestration (agent IA)

Mécanisme qui permet à un agent IA de coordonner plusieurs outils ou sous-agents pour accomplir une tâche complexe. Exemple : l'agent reçoit une demande de devis, interroge la base produits, calcule un prix, génère un PDF et l'envoie par email — en chaîne automatique.

P

Perplexity

Moteur de recherche IA qui génère des réponses synthétiques avec citations. Croissance rapide en 2025-2026, particulièrement chez les professionnels. Optimiser pour Perplexity fait partie de la stratégie GEO : contenu factuel, bien sourcé, avec données chiffrées.

Prompt Engineering

Art et science de rédiger des instructions (prompts) pour obtenir les meilleurs résultats d'un modèle IA. Compétence clé pour la configuration d'agents IA. Un bon prompt engineering peut remplacer un fine-tuning coûteux dans la majorité des cas.

R

RAG (Retrieval-Augmented Generation)

Architecture où l'agent IA récupère des informations pertinentes dans une base de données avant de générer sa réponse. Permet d'ancrer les réponses sur des faits vérifiés (vos tarifs, vos produits, vos FAQ) et réduit drastiquement les hallucinations.

RGPD (Règlement Général sur la Protection des Données)

Réglementation européenne sur la protection des données personnelles. Oblige les entreprises à informer leurs clients sur l'utilisation de leurs données, à obtenir leur consentement et à garantir leur sécurité. Les agents IA qui traitent des données clients doivent être conformes RGPD.

ROI (Retour sur Investissement)

Ratio entre le gain généré par un investissement et son coût. Pour un agent IA : (économies de temps + CA additionnel) / coût mensuel. Les agents Vulcia ciblent un ROI positif en moins de 3 mois pour le Pack Solo.

S

Schema.org

Vocabulaire standardisé de données structurées pour le web. Les balises Schema (FAQPage, Article, Product, Service…) aident Google à comprendre le contenu et peuvent déclencher des résultats enrichis (rich snippets) dans les SERP.

SEO Sémantique

Optimisation basée sur la couverture complète d'un sujet (champ sémantique) plutôt que sur la répétition d'un mot-clé. Google privilégie les contenus qui répondent à l'intention de recherche dans sa globalité. Les agents IA permettent de produire ce type de contenu à grande échelle.

SERP (Search Engine Results Page)

Page de résultats d'un moteur de recherche. En 2026, une SERP Google peut inclure : AI Overview, résultats organiques classiques, Local Pack, Shopping, People Also Ask, vidéos. L'objectif SEO est d'être présent sur plusieurs de ces surfaces simultanément.

Sitemap XML

Fichier listant toutes les URLs de votre site à destination des robots d'indexation. Un sitemap à jour avec les dates de modification accélère l'indexation de vos nouvelles pages par Google. Indispensable pour les sites de plus de 10 pages.

T

Token

Unité de base utilisée par les LLMs pour traiter le texte. Un token correspond environ à 3-4 caractères en français. Le coût d'utilisation d'un LLM est calculé en tokens (entrée + sortie). Un agent traitant 2 000 interactions/mois consomme typiquement 2 à 8 millions de tokens.

Tool Use (appel d'outil)

Capacité d'un agent IA à décider d'utiliser un outil externe (recherche web, calculatrice, API calendrier) pour répondre à une demande. La combinaison LLM + tool use est au cœur des agents IA modernes.

W

Webhook

Mécanisme d'envoi automatique de données d'un logiciel vers un autre lors d'un événement (nouvelle réservation, nouveau lead, paiement reçu). Les agents IA utilisent les webhooks pour communiquer avec votre CRM, votre logiciel de facturation ou votre agenda.

Widget Chat

Interface de conversation intégrée directement sur votre site web. C'est le canal principal de déploiement d'un agent IA. Apparaît généralement en bas à droite du site, disponible sur toutes les pages.
